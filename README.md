# Network Design & Simulation

## Overview

This project demonstrates the design and simulation of a computer network using Cisco Packet Tracer. The network was configured to provide reliable communication between devices while applying fundamental networking concepts.

## Technologies Used

- Cisco Packet Tracer
- Networking
- Routing and Switching
- IPv4 Addressing

## Features

- Network topology design
- Router and switch configuration
- IP addressing
- Network connectivity testing
- Network simulation

## Learning Outcomes

- Computer networking fundamentals
- Router and switch configuration
- IP addressing and subnetting
- Network troubleshooting

## Project Files

This repository contains:

- **Network_Hardware_Budget.pdf** – Hardware budget and equipment cost estimation for the network design.
- **Network_Design.pkt** – Cisco Packet Tracer simulation file containing the configured network topology.
